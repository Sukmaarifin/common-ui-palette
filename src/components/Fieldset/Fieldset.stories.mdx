# Fieldset

`Fieldset` is a simple wrapper component consist of [FormControl](https://material-ui.com/api/form-control/#formcontrol-api), [InputLabel](https://material-ui.com/api/input-label/#inputlabel-api), [FormHelperText](https://material-ui.com/api/form-helper-text/#formhelpertext-api), and `PriusTooltip`.
It provides control for the form elements.

## Usage

You can put native form elements (e.g. `<input />`, `<select />`, `<textarea />`) or custom form elements (e.g. `<Input />, <PriusSelect />, <PriusMultipleSelect />`) as a children.

### Basic

<Preview>
  <Story name="Basic">
    <Grid container spacing={5}>

            id="input-currency"
            name="currency"

            value="usd"
          >
            <MenuItem value={'usd'}>United States Dollar</MenuItem>
            <MenuItem value={'idr'}>Indonesian Rupiah</MenuItem>
            <MenuItem value={'jpy'} disabled>
              Japanese Yen
            </MenuItem>
          </PriusSelect>
        </Fieldset>
      </Grid>
    </Grid>
  </Story>
</Preview>
