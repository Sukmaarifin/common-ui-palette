# PriusTab

`PriusTab` is a component wrapper of [Tabs](https://material-ui.com/components/tabs/)
to organize content and easy to explore and switch between different views.
It uses `PriusTabPanel` as the [TabPanel](https://material-ui.com/api/tab-panel/). When the tab is activated, the corresponding panel is made visible, and all other panels are hidden.

### Disabled

A Tab can be disabled by setting `disabled` property values as `true`.

<Preview>
  <Story name="Disabled">
    <PriusTab
      activeTab={0}
      onChange={(event, index) => void 0}
      centered
      tabs={[
        { text: 'Tab 1', disabled: false },
        { text: 'Tab 2', disabled: true },
        { text: 'Tab 3', disabled: true },
      ]}
    />
  </Story>
</Preview>

### Scrollable

You can set scrollable tab, that will put a horizontal scrollbar on the container when the tabs get too big.
