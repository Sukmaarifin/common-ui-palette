# PriusImageUploader

`PriusImageUploader` component provides the functionality to upload an image.

## Usage

You need to use the `UPLOAD_IMAGE` mutation on `src/queries/mutation.ts` in order to upload the image and pass the uploaded image URL into the `source` props.
