# PriusInput

`PriusInput` component provides a simple wrapper of [InputBase](https://material-ui.com/api/input-base/#inputbase-api) with custom styles.

## Usage

`PriusInput` usage acts the same as [InputBase](https://material-ui.com/api/input-base/#inputbase-api) and so all props are inherited from there. Should be wrapped inside `Fieldset` or any other native component like `<label/>`.
