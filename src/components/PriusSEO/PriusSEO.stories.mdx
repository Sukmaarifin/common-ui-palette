# PriusSEO

`PriusSEO` component provides a wrapper for several input elements (i.e <LinkTo kind="Components| PriusInput">PriusInput</LinkTo> and <LinkTo kind="Components| PriusMultipleSelect">PriusMultipleSelect</LinkTo>) which data are used for the needs of SEO.

## Usage

Usually, we wrap `PriusSEO` component with <LinkTo kind="Components| Paper">Paper</LinkTo> component.

### Basic

<Preview>
  <Story name="Basic">
    <Paper padding="large">
      <PriusSEO
        data={[
          {
            title: 'PriusSEO',
            description: 'This is PriusSEO stories',
            keywords: ['SEO', 'Prius'],
            language: 'ID',
          },
        ]}
        isSlugActive={true}
        slug="test-slug"
        placeholder="Masukkan slug"
      />
    </Paper>
  </Story>
</Preview>
