# PriusToggle

`PriusToggle` is a [FormControlLabel](https://material-ui.com/api/form-control-label) that wrap [Switch](https://material-ui.com/components/switches/) component.
It can be used for boolean inputs (true / false, on / off, active / non-active).
This component have two `type` of apperance: **regular** and **table**.

## Usage

Should provide `checked` and `onChange` props.

### Regular

<Preview>
  <Story name="regular">
    {() => {
      const [checked, setChecked] = useState(false);
      return (
        <Grid container style={{ margin: 12 }}>
          <PriusToggle
            type="regular"
            checked={checked}
            onChange={() => setChecked(!checked)}
          />
          <PriusToggle
            type="regular"
            label="Some Label"
            checked={checked}
            onChange={() => setChecked(!checked)}
          />
        </Grid>
      );
    }}
  </Story>
</Preview>
