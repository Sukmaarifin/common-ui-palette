# PriusCircularProgress

`PriusCircularProgress` is a customize component from [Circular, Linear Progress](https://material-ui.com/components/progress/) Material-UI.

## Usage

By default, `isWithText` props is set to **true**. If you want to render it with without any custom text you need to set `isWithText` to false.

### Basic

<Preview>
  <Story name="Basic">
    <>
      <Grid container spacing={5} alignItems="center">
        <Grid item>
          <PriusCircularProgress isWithText={false} />
        </Grid>
        <Grid item>
          <PriusCircularProgress size={20} isWithText={false} />
        </Grid>
      </Grid>
    </>
  </Story>
</Preview>

### withText

Either way, `PriusCircularProgress` will render pure [Circular, Linear Progress](https://material-ui.com/components/progress/) Material-UI with a custom text based on the design palette.

        </Grid>

        </Grid>

</Preview>
