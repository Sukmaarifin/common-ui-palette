# Card

`Card` is a custom combination component between <LinkTo kind="Components|PriusPaper">PriusPaper</LinkTo> and <LinkTo kind="Components|PriusLabel">PriusLabel</LinkTo>.

## Usage

In this case, `icon` is required based on the design palette. If you didn't pass `onClick` props, it will automatically set `disabled` as true.
