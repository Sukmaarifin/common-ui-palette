# Tooltip

`Tooltip` can be used to display additional information on user actions (e.g. hover, focus, etc).
`Tooltip` is a simple wrapper of [MaterialUI Tooltip](https://material-ui.com/components/tooltips/) with custom styling. All of its props are inherited to the `Tooltip`

---

---

---

## Usage

Here are some common usage of `Tooltip`. You can check [MaterialUI Tooltip Doc](https://material-ui.com/components/tooltips/) for more advance usage.

### Basic

You can wrap the children with `Tooltip` which will show the `Tooltip` when users hover, focus, or tap to it.

<Preview>
  <Story name="Basic">
    <Tooltip title="Tooltip can be used to display additional information on user actions (e.g. hover, focus, etc)">
      <Button background="primary">Hover on me! I have a tooltip.</Button>
    </Tooltip>
  </Story>
</Preview>

### Placement

We can set initial `Tooltip` placement. And it will automatically adapt the placement if there is no space available.

<Preview>
  <Story name="Placement">
    <Grid>
      <Tooltip placement="top" title="I should popping up on the top!">
        <Button>Top</Button>
      </Tooltip>
      <Tooltip placement="right" title="I should popping up on the right!">
        <Button>Right</Button>
      </Tooltip>
      <Tooltip
        placement="bottom"
        title="I should popping up on the bottom!"
      >
        <Button>Bottom</Button>
      </Tooltip>
      <Tooltip placement="left" title="I should popping up on the left!">
        <Button>Left</Button>
      </Tooltip>
    </Grid>
  </Story>
</Preview>
