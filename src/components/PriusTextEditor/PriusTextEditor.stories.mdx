# PriusTextEditor

`PriusTextEditor` component is a WYSIWYG HTML editor. Fully customizable with countless features and plugins. It uses [CKEditor](https://ckeditor.com/docs/ckeditor4/latest/guide/dev_react.html) beneath the surface.

## Usage

You can customize the toolbar using the `toolbar` prop. And get the data using `event.editor.getData()`.
