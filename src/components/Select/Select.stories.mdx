# PriusSelect

`PriusSelect` component provides a simple wrapper of
[Select](https://material-ui.com/components/selects/#selects) with custom
styles. It uses `PriusInput` as the input element. All props are inherited
from [Select](https://material-ui.com/components/selects/#selects). Children
should only contains
[MenuItem](https://material-ui.com/api/menu-item/#menuitem-api).

## Usage

Because of the same props, `PriusSelect` usage is the same as [Select](https://material-ui.com/components/selects/#selects) from [Material-UI](https://material-ui.com/).
