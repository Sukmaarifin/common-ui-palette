# PriusButtonText

`PriusButtonText` The Text Buttons, Contained Buttons, Floating Action Buttons and Icon Buttons are built on top of the same component,
wraper of [ButtonBase](https://material-ui.com/api/button-base) with custom style.

### Basic

You can take advantage of this lower level component to build custom interactions.
