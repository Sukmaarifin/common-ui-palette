# PriusMultipleImage

`PriusMultipleImage` is a component that map multiple images into `PriusThumbnail` component with a [Grid](https://material-ui.com/components/grid/) wrapper.

## Usage

You can set the `isDeletable` and `handleDelete` props if you want the image to be deleted when you clicked the trashbin icon.
