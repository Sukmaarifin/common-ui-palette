# PriusFooter

This simple wrapper component is used as a footer at the end of the page.
You can put [Button](https://material-ui.com/api/button/) element as a children(s).

## Usage

```tsx
<PriusFooter>
  <Button variant="contained" type="submit" color="primary">
    Simpan
  </Button>
</PriusFooter>
```
